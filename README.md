En la cuarta semana también se solicitó un repositorio con el mismo nombre, por eso se añade el 1 al final
Nombre: Erika Tatiana España Valdez
El proyecto tiene una estructura parecida a la anteriormente usada para la semana 4, se han añadido ciertos datos y se colocó el init.py para empaquetar las carpetas. La importancia de utilziar los identificadores descriptivos se da para poder reconocer lo que se esta utilizando de forma fácil, sin estar adivinando, el tipo de datos adecuados es el que permite guardar la información tanto para productos como para clientes, por último, las listas están para mostrar varios productos o clientes en un solo llamado.
